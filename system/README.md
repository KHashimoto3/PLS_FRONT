# ファイル構成

＜ src フォルダ内＞  
App.vue：システムのトップページ  
CodingForm.vue：実際にコーディングを行う「コーディングフォーム」  
＜ src → components ディレクトリ内＞  
CodeRun.vue：外部 API を用いて、システムで記述されたプログラムを実行できるコンポーネント  
CodingForm.vue から利用
